# JITS Self Attendance

A browser-based self attendance tracker for Jyothishmathi Institute of Technology and Science, B.Tech 4th Year 1st Semester.

## Features

- Student login with name and hall ticket number
- Branch and section timetable selection
- Daily present/absent marking
- Lab attendance by hours
- Holiday, mid exam, and lab exam day support
- Subject-wise attendance percentage
- Overall attendance percentage
- Past day review and editing
- Local browser storage

## Use

Open `index.html` in a browser, or publish this repository with GitHub Pages.

All attendance data is stored only in the user's browser on their own device.
