# Project Structure

This export is for learning and downloading.

## Files

- `public/index.html`  
  The complete working app. Open this file in a browser.

- `learning/app-complete.html`  
  A copy of the complete app in one file.

- `learning/styles.css`  
  The CSS extracted from `index.html`. This controls layout, colors, buttons, tables, and responsive design.

- `learning/app.js`  
  The JavaScript extracted from `index.html`. This controls login, timetable, attendance saving, percentages, past days, and backups.

- `README.md`  
  Basic project explanation for GitHub.

- `.gitignore`  
  Tells Git which local folders not to upload.

## Why GitHub Shows Only 3 Files

The live app was intentionally published as a simple single-file website:

- `.gitignore`
- `README.md`
- `index.html`

That is enough for GitHub Pages. The app does not need extra files to run.
