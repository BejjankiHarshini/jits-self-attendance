# How The App Works

## HTML

The HTML creates the screens:

- login/register screen
- student dashboard
- daily attendance class list
- subject percentage table
- past days history
- timetable editor

## CSS

The CSS controls the design:

- sidebar
- cards
- buttons
- mobile layout
- tables
- percentage colors

## JavaScript

The JavaScript controls the app logic:

- saves students in browser local storage
- loads the correct branch and section timetable
- marks present/absent
- calculates lab hours
- calculates subject-wise percentage
- calculates overall percentage
- stores past day attendance
- exports/imports backup JSON

## Storage

The app stores data in the browser using `localStorage`.

This means every user has their own data on their own device.
